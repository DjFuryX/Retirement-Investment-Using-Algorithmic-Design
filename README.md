
# Getting Started

Retirement Investment Optimization Using
Algorithmic Design


## Build Instructions  

Ensure you have a JDK installed on your  system. You can check by running:
>```java -version```
otherwise install by visiting
```The Official Java Website ```
## Command Line
 - clone the repo or download the zip file
 - navigate to the root of the **src** folder in the terminal
 - execute ```javac RetirementInvestment.java``` to compile
 - execute ```java RetirementInvestment``` to run program

## Visual Studio Code
 - install a java compiler extension
 - install Github Pull Request
 - Clone git Repository 
 - compile in editor (normally F5)

# Running the Program
 A already compiled executeable is included in the root folder
 click to open the program

The menu is command line driven simply enter the number correspondig to the disered option and enter ```0``` to exit
